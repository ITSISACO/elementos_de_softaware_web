respuestas

1. a.  Un Servidor DNS (Domain Name System).
Se encarga de traducir los nombres de dominio (por ejemplo, google.com) en direcciones IP que las computadoras entienden.
Sin DNS, tendrías que memorizar números como 142.250.190.78 para acceder a un sitio.

b. La dirección IP es un número único que identifica a cada dispositivo conectado a Internet.

c. Un protocolo es un conjunto de reglas que permiten la comunicación entre dispositivos.  Define cómo se envían y reciben los datos.
Ejemplos:
HTTP → para páginas web
HTTPS → versión segura de HTTP
FTP → para transferir archivos

d. Es el nombre fácil de recordar que usamos para acceder a sitios web

e. Es la ruta específica dentro de un sitio web que indica dónde está un recurso. Es como la ubicacion exacta dentro de una carpeta 

2. a. Diseñador UI/UX : Se encarga de la experiencia y apariencia del usuario.
UI (User Interface): diseño visual 
UX (User Experience): facilidad de uso y navegación

b. Administra servidores y servicios en la nube. Sus funciones son:
Configurar servidores
Garantizar disponibilidad
Escalar recursos
Seguridad

c. Gestiona las bases de datos. Sus funciones son:
Crear y mantener bases de datos
Optimizar consultas
Hacer copias de seguridad
Garantizar seguridad de datos

d. Coordina todo el proyecto. Sus funciones son: 
Planificar tareas
Gestionar tiempos y recursos
Coordinar el equipo
Asegurar cumplimiento de objetivos

3.  Un lenguaje es Turing completo cuando puede resolver cualquier problema computacional si tiene suficiente tiempo y memoria. Ejm : JavaScript

4. HTML y CSS no son lenguajes de programación porque no pueden ejecutar lógica ni algoritmos. HTML define la estructura de la página y CSS su apariencia, pero no permiten tomar decisiones ni repetir procesos como sí lo hacen los lenguajes de programación.

5. Una etiqueta (tag) es un elemento que se usa para definir partes del contenido en una página web.
